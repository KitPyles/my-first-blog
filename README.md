# My First Blog
This is my first attempt at building a blog from scratch using Python and the Django Framework. I followed the [DjangoGirls tutorial](https://tutorial.djangogirls.org/en/).
## Installation
There is no installation. You can follow the tutorial linked above to create your own blog from scratch, or fork this repo to get a working model to start.
## Usage
It's a blog. You write, edit, and delete posts.
